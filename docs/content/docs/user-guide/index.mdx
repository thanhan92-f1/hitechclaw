## Prerequisites

Before diving in, make sure you have:
1. xClaw running via `docker compose up --build`
2. Accessed the web interface at `http://localhost:3000`
3. Logged in with your credentials (default: `admin@xclaw.io` / `password123`)

## Navigation

The left sidebar displays the xClaw logo with "v2" badge and organizes features into groups:

- **MAIN** — Dashboard and Chat
- **KNOWLEDGE** — Knowledge Base and RAG Search
- **DOMAINS** — Domain pack selector (13 industry domains with expandable list)
- **AGENTS** — Agent management, channels, workflows, marketplace, analytics, and builder tools
- **TOOLS** — Ollama Models, ML/AutoML, Medical Tools, MCP Servers
- **SYSTEM** — Settings, user profile, and logout
