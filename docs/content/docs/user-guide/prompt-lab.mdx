# Prompt Lab

![Prompt Lab](/screenshots/11-prompt-lab.png)

The Prompt Lab provides ready-made agent templates and an interactive playground for testing prompts with different models.

---

## Templates

Six pre-built templates for common use cases. Each template includes a curated persona, recommended skills, and optimized settings.

### Customer Support
A friendly support agent for handling common customer inquiries.
- **Tags**: support, customer-facing
- **Best for**: FAQ handling, issue resolution, general assistance
- **Recommended temperature**: 0.7
- **Key traits**: Empathetic, patient, solution-oriented

### Code Assistant
A technical coding assistant for developers.
- **Tags**: developer, technical
- **Best for**: Code generation, debugging, architecture review
- **Recommended temperature**: 0.3
- **Key traits**: Precise, thorough documentation, follows best practices

### Data Analyst
An analytics expert for data interpretation and insights.
- **Tags**: analytics, data
- **Best for**: Data analysis, SQL queries, visualization recommendations
- **Recommended temperature**: 0.5
- **Key traits**: Methodical, data-driven, clear explanations

### Content Writer
A creative writer for marketing and content production.
- **Tags**: marketing, content
- **Best for**: Blog posts, social media, ad copy, email campaigns
- **Recommended temperature**: 0.8
- **Key traits**: Creative, engaging, brand-aware

### Healthcare Assistant
A medical knowledge assistant (non-diagnostic).
- **Tags**: healthcare, medical
- **Best for**: Medical information, drug lookups, clinical documentation
- **Recommended temperature**: 0.2
- **Key traits**: Evidence-based, cautious, always recommends professional consultation

### Multilingual Agent
An agent that auto-detects language and responds in kind.
- **Tags**: multilingual, general
- **Best for**: International support, translation, cross-language communication
- **Recommended temperature**: 0.6
- **Key traits**: Accurate language detection, culturally aware, fluent responses

## Using a Template

1. Click on a template card
2. Review the pre-configured persona, skills, and settings
3. Click **Use Template** to create a new agent based on this template
4. Customize any fields as needed
5. Save the agent

Templates serve as starting points — you can modify every aspect after applying.

---

## Playground

Switch to the **Playground** tab for interactive prompt testing.

### Features
- **Model Selector**: Choose which LLM to test with
- **System Prompt Editor**: Write and edit system prompts in real-time
- **Temperature Slider**: Adjust creativity on the fly
- **Chat Interface**: Send messages and see responses immediately
- **Token Count**: Monitor input and output token usage
- **Response Time**: Track how long each response takes

### Use Cases
- **A/B Test Prompts**: Compare different system prompts with the same input
- **Model Comparison**: Test the same prompt across different models
- **Temperature Tuning**: Find the optimal temperature for your use case
- **Prompt Engineering**: Iterate on prompts without creating full agents

---

## Tips

- Use the Playground to prototype before building a full agent
- Compare at least 2–3 temperature values when tuning
- Test with adversarial inputs to check agent safety
- Export successful prompts from the Playground directly into new agents
