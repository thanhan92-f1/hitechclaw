# Domain Store

![Domain Store](/screenshots/17-domains.png)

The Domain Store lets you browse and install 13 AI domain specializations. Each domain comes with pre-configured skills, system prompts, and tools optimized for a specific industry.

---

## Overview

The Domain Store header shows:
- **Total domains**: 13 available
- **Installed count**: Domains currently active
- **Total skills**: Sum of all skills across domains (24 skills)

### Filter Tabs
- **All (13)** — Show all domains
- **Installed** — Show only installed domains
- **Available** — Show domains not yet installed

### Search
Use the search bar to find domains by name or description.

## Installing a Domain

### Step-by-Step
1. Navigate to **Domain Hub** (sidebar → All Domains → Domain Hub)
2. Browse the domain cards or use the search bar
3. Click **Install** on the desired domain card
4. The domain's skills are automatically loaded
5. Installed domains show a green **Installed** badge

### After Installation
- The domain appears in the Chat page's domain selector
- Its skills become available for agent assignment
- Domain-specific system prompts are activated when selected in chat

---

## Domain Details

Click **Details →** on any domain card to view:
- Full description
- List of included skills with descriptions
- Configuration options
- Usage examples

---

## Sidebar Navigation

When domains are expanded in the sidebar, you see:
- **Domain Hub** — The main domain store page
- Individual domain entries with skill counts
- Click a domain name to view its detail page
- **+5 more...** link if more than 8 domains are installed

---

## Using Domains in Chat

1. Open a new chat conversation
2. Click the **Select Domain** dropdown
3. Choose a domain (e.g., "Healthcare & Medical")
4. The AI's system prompt and skills are automatically adjusted for that domain
5. Start chatting — the AI will use domain-specific knowledge and tools

---

## Tips

- Start with **General Purpose** — it covers most everyday tasks
- Install domain-specific packs only when needed to keep the interface clean
- Combine domains with marketplace skills for even more specialized agents
- Each domain's skills can be individually assigned to agents in the Agent Builder
