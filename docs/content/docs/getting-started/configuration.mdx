xClaw is configured through environment variables in the `.env` file.

## Environment Setup

Copy the example environment file:

```bash
cp .env.example .env
```

## LLM Configuration

### Provider Selection

```ini title=".env"
LLM_PROVIDER=openai          # openai | anthropic | ollama
```

### OpenAI

```ini title=".env"
LLM_PROVIDER=openai
OPENAI_API_KEY=sk-...
LLM_MODEL=gpt-4o-mini        # gpt-4o, gpt-4o-mini
LLM_TEMPERATURE=0.7
LLM_MAX_TOKENS=4096
```

### Anthropic Claude

```ini title=".env"
LLM_PROVIDER=anthropic
ANTHROPIC_API_KEY=sk-ant-...
LLM_MODEL=claude-3-haiku-20240307  # claude-3-sonnet, claude-3-haiku
```

### Ollama (Local)

```ini title=".env"
LLM_PROVIDER=ollama
OLLAMA_URL=http://localhost:11434
LLM_MODEL=llama3              # llama3, mistral, phi3
```

<Callout type="info">
Ollama uses the OpenAI-compatible API format. Install Ollama from [ollama.ai](https://ollama.ai) and pull your desired model: `ollama pull llama3`
</Callout>

## Provider Comparison

| Provider | Models | Notes |
|:---:|---|---|
| **OpenAI** | gpt-4o-mini, gpt-4o | Requires `OPENAI_API_KEY` |
| **Anthropic** | claude-3-haiku, claude-3-sonnet | Requires `ANTHROPIC_API_KEY` |
| **Ollama** | llama3, mistral, phi3 | Local, free |

## Channel Configuration

### Telegram

```ini title=".env"
TELEGRAM_BOT_TOKEN=your-telegram-bot-token
```

### Discord

```ini title=".env"
DISCORD_BOT_TOKEN=your-discord-bot-token
```

<Callout type="info">
Add bot tokens to `.env` to auto-enable channels. Remove the token to disable.
</Callout>

## Server Configuration

```ini title=".env"
PORT=18789                    # API server port
WEB_PORT=3000                 # Web UI port
NODE_ENV=development          # development | production
```

## CLI Usage

```bash
# Start the gateway
npx tsx packages/cli/src/index.ts gateway

# Chat with the agent
npx tsx packages/cli/src/index.ts chat "Hello, what can you do?"

# List skills
npx tsx packages/cli/src/index.ts skills list

# System health check
npx tsx packages/cli/src/index.ts doctor
```
