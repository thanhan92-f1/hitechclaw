## Setup

1. Create a GitHub Personal Access Token (PAT) with `repo` scope
2. Add the token in **Settings** → **Integrations** → **GitHub**

## Available Tools

| Tool | Description |
|------|-------------|
| `list_repos` | List repositories for the authenticated user |
| `create_issue` | Create a new issue in a repository |
| `list_issues` | List issues with optional filters (state, labels) |
| `create_pull_request` | Create a pull request from a branch |
| `get_file_contents` | Read file contents from a repository |

## Example Usage

> "Create an issue in my hitechclaw repo titled 'Add dark mode support' with label 'enhancement'"

The agent will use `create_issue` to create the issue with the specified title and labels.
