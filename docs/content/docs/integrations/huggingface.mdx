---
title: HuggingFace
description: Run ML model inference and embeddings via HuggingFace API.
---

## Setup

1. Get an API token from [HuggingFace Settings](https://huggingface.co/settings/tokens)
2. Add the token in **Settings** → **Integrations** → **HuggingFace**

## Available Tools

| Tool | Description |
|------|-------------|
| `inference` | Run inference on a HuggingFace model |
| `list_models` | Browse available models by task |
| `list_datasets` | List datasets on the Hub |
| `get_model_info` | Get metadata about a specific model |
| `text_embedding` | Generate text embeddings for RAG |
