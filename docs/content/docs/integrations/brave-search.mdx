---
title: Brave Search
description: Web and news search powered by Brave Search API.
---

## Setup

1. Get an API key from [Brave Search API](https://brave.com/search/api/)
2. Add the key in **Settings** → **Integrations** → **Brave Search**

## Available Tools

| Tool | Description |
|------|-------------|
| `web_search` | Search the web with a query string |
| `news_search` | Search recent news articles |
