title: Notion
description: Search, read, and create Notion pages and databases through your AI agent.
---

## Setup

1. Create an internal integration at [Notion Developers](https://www.notion.so/my-integrations)
2. Share the target pages/databases with your integration
3. Add the secret in **Settings** → **Integrations** → **Notion**

## Available Tools

| Tool | Description |
|------|-------------|
| `search` | Search across all shared Notion pages |
| `get_page` | Retrieve a specific page and its content |
| `create_page` | Create a new page in a parent page or database |
| `query_database` | Query a Notion database with optional filters |
