title: Weights & Biases
description: Track ML experiments and artifacts through your AI agent.
---

## Setup

1. Get an API key from [W&B Settings](https://wandb.ai/settings)
2. Add the key in **Settings** → **Integrations** → **W&B**

## Available Tools

| Tool | Description |
|------|-------------|
| `log_run` | Create and log a new experiment run |
| `list_runs` | List runs in a project |
| `get_run` | Get details of a specific run |
| `log_artifact` | Log an artifact (dataset, model) to a run |
| `create_report` | Create a W&B report |
