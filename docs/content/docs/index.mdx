Welcome to the **HiTechClaw** documentation. HiTechClaw is an open-source AI Agent Platform built with TypeScript that lets you build, deploy, and manage AI agents with ease.

## What is HiTechClaw?

HiTechClaw provides a complete toolkit for creating AI-powered applications:

- **Pluggable Skill System** — 11+ pre-built skill packs for programming, healthcare, analytics, and more
- **Plugin Architecture** — Official plugins (TeeForge.AI, Healthcare) maintained as [git submodules](https://github.com/thanhan92-f1/hitechclaw-plugins)
- **Visual Workflow Builder** — Drag-and-drop canvas with 16 node types
- **Multi-LLM Support** — OpenAI, Anthropic, DeepSeek, xAI, OpenRouter, Perplexity, Google, Groq, Mistral, Ollama
- **Agent Hub Marketplace** — Browse, install, and manage AI agents
- **Multi-Channel** — Telegram, Discord, Slack, WhatsApp, Zalo OA, Microsoft Teams, WebChat, Webhook
- **Semantic Memory** — Vector-based long-term memory with cosine similarity
